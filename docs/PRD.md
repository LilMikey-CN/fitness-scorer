# 产品需求文档（PRD）

## 1. 标准来源

《国民体质测定标准》成年人部分，国家体育总局群体司，2003 年。
适用人群：**20–59 周岁**中国成年人，按性别和 5 岁一组分为 16 个组别。

---

## 2. 年龄计算规则

```
测试时已过当年生日：年龄 = 测试年 - 出生年
测试时未过当年生日：年龄 = 测试年 - 出生年 - 1
```

年龄组划分（每 5 岁一组）：
`20–24 | 25–29 | 30–34 | 35–39 | 40–44 | 45–49 | 50–54 | 55–59`

---

## 3. 测试指标与适用范围

| 指标 | 20–39 岁 | 40–59 岁 | 单位 |
|------|----------|----------|------|
| 身高标准体重 | ✓（男+女） | ✓（男+女） | cm / kg |
| 肺活量 | ✓（男+女） | ✓（男+女） | mL |
| 台阶指数 | ✓（男+女） | ✓（男+女） | 无量纲 |
| 握力 | ✓（男+女） | ✓（男+女） | kg |
| 俯卧撑次数 | ✓（**仅男**） | — | 次 |
| 1分钟仰卧起坐 | ✓（**仅女**） | — | 次 |
| 纵跳 | ✓（男+女） | — | cm |
| 坐位体前屈 | ✓（男+女） | ✓（男+女） | cm（可负值）|

**每组有效指标数：**
- 20–39 岁男性：9 项

- 40–59 岁男/女：7 项

## 4. 单项评分规则

### 4.1 身高标准体重（特殊规则）

只有 1 分、3 分、5 分三个级别（无 2 分、4 分），评分呈"钟型"分布：

```
weight < b1             → 1 分（偏轻）
b1 ≤ weight < b3a       → 3 分（轻）
b3a ≤ weight < b3b      → 5 分（正常）
b3b ≤ weight < b1b      → 3 分（重）
weight ≥ b1b            → 1 分（偏重）
```

查表按：**年龄段（20-29/30-39/40-49/50-59）× 性别 × 身高段（1cm 为一档）**

### 4.2 其他升序指标

肺活量、台阶指数、握力、俯卧撑、仰卧起坐、纵跳、坐位体前屈、闭眼单脚站立。
分数越高越好，使用 [min1, min2, min3, min4, min5] 五个阈值：

```
value < min1        → 0 分（无分，不计入综合评级）
min1 ≤ value < min2 → 1 分
min2 ≤ value < min3 → 2 分
min3 ≤ value < min4 → 3 分
min4 ≤ value < min5 → 4 分
value ≥ min5        → 5 分
```

### 4.3 降序指标（选择反应时）

时间越短越好，使用 [max1, max2, max3, max4, max5] 五个阈值：

```
value > max1                  → 0 分（无分）
max2 < value ≤ max1           → 1 分
max3 < value ≤ max2           → 2 分
max4 < value ≤ max3           → 3 分
max5 < value ≤ max4           → 4 分
value ≤ max5                  → 5 分
```

---

## 5. 综合评级规则

**任意一项得分为 0，不给出综合评级（输出 null/"—"）。**

| 年龄段 | 一级（优秀） | 二级（良好） | 三级（合格） | 四级（不合格） |
|--------|------------|------------|------------|--------------|
| 20–39 岁 | > 33 分 | 30–33 分 | 23–29 分 | < 23 分 |
| 40–59 岁 | > 26 分 | 24–26 分 | 18–23 分 | < 18 分 |

## 6. 功能需求

### 6.1 手动录入模式

- 界面分两步：① 填写基本信息 → ② 填写各项指标

**步骤一 基本信息字段：**
| 字段 | 类型 | 说明 |
|------|------|------|
| 姓名 | 文本 | 可选，仅用于显示 |
| 性别 | 下拉 | 男 / 女 |
| 出生日期 | 日期选择 | 用于精确计算周岁年龄 |
| 测试日期 | 日期选择 | 默认为当天，用于与出生日期联算年龄 |

**年龄计算方式（来自 §2）：**
```
age = testYear - birthYear
若测试日期 < 当年生日 → age -= 1
```
系统在步骤一提交时自动计算周岁年龄，不需要用户手动填写年龄数字。

- 根据性别和年龄段动态显示/隐藏字段（如 40 岁男性不显示俯卧撑和纵跳）
- 实时展示每项原始数值 → 该项得分（1–5 分或"无分"）
- 右侧汇总面板：各项得分 + 总分 + 综合评级（等级 + 描述）
- 可打印/导出单人结果（可选 v1.0 后实现）

### 6.2 批量导入模式

- 支持 `.xlsx`、`.xls`、`.csv` 文件
- 点击"选择文件"后解析，展示预览表格（前 10 行 + 总行数）
- 校验错误（字段缺失、数值超范围等）以行为单位标红提示，不中断处理
- 点击"开始计算"后逐行评分，进度条显示
- 结果可导出为 `.xlsx`（在原始数据右侧追加各项得分列 + 综合评级列）
- 导出时弹出 Windows 文件保存对话框

### 6.3 公共 UI 要求

- 全中文界面
- 支持 Windows 高 DPI（至少 125%、150% 缩放下不错位）
- 错误提示需明确（如"身高超出范围（144–199cm）"而非通用"输入无效"）

---

## 7. 非功能需求

- **无网络依赖**：完全离线运行
- **无需安装 Node.js/运行时**：最终交付物为独立 .msi
- **首次启动 < 3 秒**
- **批量处理 1000 行 < 10 秒**
- **数据不持久化**：关闭窗口后数据不保留（v1.0 不需要历史记录）

在 `src/data/scoringTables.ts` 中以注释标注。主要问题：

| 30-39 男，167cm | 5分区间"54.7–70.6"与3分区间"71.7–75.8"有空隙 | 保留原值 |
| 30-39 女，167cm | 3分区间下限"82.1"疑为"72.1"印刷错误 | 修正为 72.1 |
| 闭眼单脚站立，20-24 男 | "18–14"疑为"18–41" | 修正为 41 |
| 闭眼单脚站立，20-24 女 | 3/4 分区间有重叠（34/36）| 采用 3 分上限 33 |

| 50-59 男，146cm | 1分"<40.1"与3分起始"41.0"之间有空隙 | 1分上限取 41.0 |
