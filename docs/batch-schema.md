# 批量导入/导出字段规范

## 输入文件格式（Excel / CSV）

第一行为**表头**，字段名不区分大小写，支持中英文（见下表）。
字段顺序不要求固定，系统按字段名匹配。

---

## 输入字段定义

| 列名（中文） | 列名（英文别名） | 类型 | 必填 | 说明 |
|------------|----------------|------|------|------|
| 姓名 | name | 字符串 | 否 | 仅用于输出，不影响计算 |
| 性别 | gender / sex | 整数 | **是** | **1 = 男，0 = 女** |
| 年龄 | age | 整数 | **是** | 周岁（20–59），或提供出生年替代 |
| 出生年 | birth_year | 整数 | 否 | 若无年龄列则必填，测试年由系统时钟获取 |
| 身高 | height | 浮点 | **是** | 单位：**cm**，精度 0.1，范围见下 |
| 体重 | weight | 浮点 | **是** | 单位：**kg**，精度 0.1，范围见下 |
| 肺活量 | lung_capacity / vital_capacity | 整数 | **是** | 单位：mL |
| 台阶指数 | step_index | 浮点 | **是** | 计算公式：运动持续时间(s) / (3次脉搏之和) × 100 |
| 握力 | grip_strength | 浮点 | **是** | 单位：kg，精度 0.1 |
| 俯卧撑 | pushups / push_ups | 整数 | 条件必填 | **仅男性 20–39 岁**，其他行留空 |
| 仰卧起坐 | situps / sit_ups | 整数 | 条件必填 | **仅女性 20–39 岁**，其他行留空 |
| 纵跳 | vertical_jump | 浮点 | 条件必填 | **仅 20–39 岁**，单位：cm，精度 0.1 |
| 坐位体前屈 | sit_and_reach | 浮点 | **是** | 单位：cm，精度 0.1，**可为负值** |
| 选择反应时 | reaction_time | 浮点 | **是** | 单位：秒，精度 0.01 |
| 闭眼单脚站立 | single_leg_stand | 浮点 | **是** | 单位：秒，精度 0.1 |

---

## 条件必填说明

- **俯卧撑**：年龄 20–39 且性别=1（男）时必填；其他情况留空（系统自动跳过）
- **仰卧起坐**：年龄 20–39 且性别=0（女）时必填；其他情况留空
- **纵跳**：年龄 20–39 时必填（男女均需）；40 岁以上留空

## 输出文件格式

在原始列右侧**追加**以下列（保留原始数据，方便核对）：

| 追加列名 | 内容 |
|---------|------|
| 年龄组 | 如 `30-34岁` |
| 身高体重_分 | 1 / 3 / 5 / 无分 |
| 肺活量_分 | 1–5 / 无分 |
| 台阶指数_分 | 1–5 / 无分 |
| 握力_分 | 1–5 / 无分 |
| 俯卧撑_分 | 1–5 / 无分 / — |
| 仰卧起坐_分 | 1–5 / 无分 / — |
| 纵跳_分 | 1–5 / 无分 / — |
| 坐位体前屈_分 | 1–5 / 无分 |
| 选择反应时_分 | 1–5 / 无分 |
| 闭眼单脚站立_分 | 1–5 / 无分 |
| 总分 | 数字（各单项得分之和）|
| 综合评级 | 一级(优秀) / 二级(良好) / 三级(合格) / 四级(不合格) / — |
| 备注 | 错误信息或警告，正常行为空 |

"—" 表示该项不适用于该年龄/性别组合。
"无分" 表示测量值超出评分范围（或未填写必填字段）。

## CSV 编码处理

```typescript
// 读取时：自动探测 BOM，无 BOM 则优先尝试 UTF-8，失败则 fallback GBK
import * as XLSX from 'xlsx';

function readWorkbook(buffer: ArrayBuffer) {
  // SheetJS 会自动检测 UTF-8 BOM
  // 对于中文 Windows 导出的无 BOM GBK 文件，指定 codepage
  try {
    return XLSX.read(buffer, { type: 'array' });
  } catch {
    return XLSX.read(buffer, { type: 'array', codepage: 936 }); // GBK
  }
}

// 写出时：始终 UTF-8（xlsx 格式内置处理）
function writeOutput(wb: XLSX.WorkBook, filePath: string) {
  XLSX.writeFile(wb, filePath); // xlsx 默认 UTF-8
}
```
