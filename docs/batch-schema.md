# 批量导入/导出字段规范

## 输入文件格式（Excel / CSV）

第一行为**表头**，字段名不区分大小写，支持中英文（见下表）。
字段顺序不要求固定，系统按字段名匹配。

---

## 输入字段定义

| 列名（中文） | 列名（英文别名） | 类型 | 必填 | 说明 |
|------------|----------------|------|------|------|
| 姓名 | name | 字符串 | 否 | 仅用于输出，不影响计算 |
| 性别 | gender / sex | 文本或整数 | **是** | **1/男/M = 男，0/女/F = 女** |
| 年龄 | age | 整数 | **是** | 周岁（20–59），或提供出生年替代 |
| 出生年 | birth_year | 整数 | 否 | 若无年龄列则必填，测试年由系统时钟获取 |
| 身高 | height | 浮点 | **是** | 单位：**cm**，精度 0.1 |
| 体重 | weight | 浮点 | **是** | 单位：**kg**，精度 0.1 |
| 肺活量 | lung_capacity / vital_capacity | 整数 | 否 | 单位：mL；缺省时该项输出 "—" |
| 台阶指数 | step_index | 浮点 | 否 | 公式：运动持续时间(s) / (3次脉搏之和) × 100；缺省时输出 "—" |
| 握力 | grip_strength | 浮点 | 否 | 单位：kg；缺省时输出 "—" |
| 俯卧撑 | pushups / push_ups | 整数 | 条件可选 | **仅男性 20–39 岁**；其他行留空；缺省时输出 "—" |
| 仰卧起坐 | situps / sit_ups | 整数 | 条件可选 | **仅女性 20–39 岁**；其他行留空；缺省时输出 "—" |
| 纵跳 | vertical_jump | 浮点 | 条件可选 | **仅 20–39 岁**，单位：cm；缺省时输出 "—" |
| 坐位体前屈 | sit_and_reach | 浮点 | 否 | 单位：cm，**可为负值**；缺省时输出 "—" |
| 选择反应时 | reaction_time | 浮点 | 否 | 单位：秒；缺省时输出 "—" |
| 闭眼单脚站立 | single_leg_stand | 浮点 | 否 | 单位：秒；缺省时输出 "—" |

**必填字段**：性别、年龄（或出生年）、身高、体重。只要这四项完整，行就能成功解析。
其余测量字段全部可选，缺省时该项得分显示 "—"，不影响已有字段的评分计算。

---

## 条件可选字段说明

- **俯卧撑**：年龄 20–39 且性别=男时有效；其他组合留空（系统自动跳过）
- **仰卧起坐**：年龄 20–39 且性别=女时有效；其他组合留空
- **纵跳**：年龄 20–39 时有效（男女均需）；40 岁以上留空

---

## 总分与综合评级计算规则

仅当**所有适用字段均有数据**时才计算总分和综合评级：
- 适用字段：本人年龄/性别组合下需要测量的全部指标
- 若任意适用字段缺省 → 总分/综合评级输出 "—"
- 若任意适用字段得分为 0（无分）→ 总分正常显示，综合评级输出 "—"

## 输出文件格式

在原始列右侧追加**两行表头 + 得分列**，分为四类指标分组：

### 第一行（分组标签，合并单元格）

| 分组 | 覆盖列 |
|------|--------|
| （无标签） | 年龄组 |
| 形态指标 | 身高体重_分 |
| 机能指标 | 肺活量_分、台阶指数_分 |
| 体能指标 | 握力_分、俯卧撑_分、仰卧起坐_分、纵跳_分 |
| 柔韧灵敏平衡指标 | 坐位体前屈_分、选择反应时_分、闭眼单脚站立_分 |
| （无标签） | 总分、综合评级、备注 |

### 第二行（字段名）

年龄组、身高体重_分、肺活量_分、台阶指数_分、握力_分、俯卧撑_分、仰卧起坐_分、纵跳_分、坐位体前屈_分、选择反应时_分、闭眼单脚站立_分、总分、综合评级、备注

### 单元格值说明

| 值 | 含义 |
|----|------|
| 1–5 | 得分 |
| 无分 | 数据已提供，但测量值超出评分范围 |
| — | 数据未提供 或 该指标不适用于此年龄/性别组合 |

## CSV 编码处理

```typescript
// 读取：自动检测 UTF-8 BOM；无 BOM 时尝试 UTF-8，失败则 fallback GBK
function readWorkbook(buffer: ArrayBuffer) {
  try {
    return XLSX.read(buffer, { type: 'array' });
  } catch {
    return XLSX.read(buffer, { type: 'array', codepage: 936 }); // GBK
  }
}
// 写出：始终 UTF-8（xlsx 格式内置处理）
```
